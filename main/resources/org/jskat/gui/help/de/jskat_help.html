JSkat Hierarchie

Bei JSkat ist jedes einzelne Skatspiel in eine strikte Hierarchie eingebunden, welche die Situation in der Wirklichkeit widerspiegel.

  • Zuerst gibt es den Skattisch, an dem die Skatspiele gespielt werden. An einem Tisch können drei Spieler sitzen. Jeder Tisch kann benannt werden.
  • An einem Tisch kann eine Skatserie gestartet werden. Am Anfang müssen die Skatspieler festgelegt werden. Ein Spieler kann eine KI oder ein menschlischer Spieler sein. Über das Internet können weitere Spieler (KIs oder Menschen) am Tisch sitzen. Jeder Spieler hat seinen eigenen Namen. Eine Skatserie kann auf eine bestimmte Anzahl Spiele begrenzt werden. Schließlich werden für jede Skatserie die erlaubten Skatregeln (z.B. offizielle Tunierregeln, Kneipenregeln oder individuelle Regeln) festgelegt.
  • In einer Skatserie werden nacheinander die Skatspiele durchgeführt. Obwohl man mehrere Spiele zu einer Runde zusammenfassen könnte, haben wir diese zusätzliche Hierarchieebene nicht umgesetzt. Je nach festgelegten Skatregeln für die Skatserie hat jedes Spiel die gleichen Regeln.

Skatspiel

Ein Skatspiel besteht aus folgenden Phasen:

  1. Karten geben
  2. Reizen
  3. Skateinsicht und Drücken (optional)
  4. Spielansage
  5. Ausspielen (bis zu 10 Stiche)
  6. Spielende mit Auswertung
